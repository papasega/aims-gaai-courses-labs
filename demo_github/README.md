# Demo GitHub for Labs

This folder contains a short notebook to show students the GitHub Classroom workflow for course labs.

Open:

```bash
jupyter notebook demo_git_github_labs.ipynb
```

Main workflow covered:

```bash
git clone <your-github-classroom-repo-url>
cd <repo-folder>
git status
git add .
git commit -m "lab_s5: complete prompting demo"
git push
```

The notebook also includes:

- how to configure Git identity;
- how to pull updates from the instructor repository;
- how to check remote URLs;
- how to verify that a lab was pushed correctly.
